## Executive Overview

Extensão do acesapp para incluir um sistema de comando central para operações de Direct Response. O MOS fornece visibilidade 360° da operação com entrada manual de dados — sem integrações de API no MVP.

**Decisões arquiteturais:**
- Nova tabela `assets` (Asset Registry — propósito diferente de `platforms`)
- Extend `ad_performance` com `product` + `profit` (Cash Flow)
- Extend `ad_campaigns` com `product` + `country` (Campaign Control)
- Zero breaking changes — tudo aditivo

## Acceptance Criteria do Epic

- [ ] Usuário consegue cadastrar todos os ativos da operação (domínios, pixels, contas, etc.)
- [ ] Usuário consegue registrar spend + revenue diário por produto
- [ ] Usuário consegue ver P&L consolidado por período em < 3 cliques
- [ ] Usuário consegue registrar e acompanhar campanhas com ROAS calculado
- [ ] Todos os dados persistem via Supabase com padrão existente do projeto
- [ ] Novas páginas acessíveis via sidebar + rotas protegidas
- [ ] Zero tabelas ou componentes existentes quebrados
