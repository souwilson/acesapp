## Dev Notes

### Stack e padrões (NÃO inventar novos padrões — seguir os existentes)

**Hook pattern** — copiar estrutura de `src/hooks/usePlatforms.ts` ou similar:
```typescript
import { useQuery, useMutation, useQueryClient } from '@tanstack/react-query';
import { supabase } from '@/integrations/supabase/client';
import { toast } from 'sonner';

export const useAssets = () => {
  return useQuery({
    queryKey: ['assets'],
    queryFn: async () => {
      const { data, error } = await supabase.from('assets').select('*').order('created_at', { ascending: false });
      if (error) throw error;
      return data;
    }
  });
};
```

**Form pattern** — usar React Hook Form + Zod + shadcn Form:
```typescript
import { useForm } from 'react-hook-form';
import { zodResolver } from '@hookform/resolvers/zod';
import { z } from 'zod';
import { Form, FormField, FormItem, FormLabel, FormControl, FormMessage } from '@/components/ui/form';
```

**Dialog pattern** — ver `src/components/forms/PlatformFormDialog.tsx` como referência.

**Imports:** sempre usar alias `@/` (ex: `@/components/ui/button`, `@/hooks/useAssets`)

**Notificações:** usar Sonner (`import { toast } from 'sonner'`) — NÃO usar useToast

**Tipos:** a tabela `assets` não está em `src/integrations/supabase/types.ts` ainda (será gerada após migration). Usar tipo local temporário:
```typescript
export interface Asset {
  id: string;
  product: string;
  asset_type: 'domain' | 'page' | 'pixel' | 'ad_account' | 'gateway' | 'checkout' | 'email' | 'other';
  name: string;
  country: string | null;
  status: 'online' | 'paused' | 'banned' | 'dead';
  link_or_id: string | null;
  notes: string | null;
  last_checked: string | null;
  created_at: string;
  updated_at: string;
}
```
